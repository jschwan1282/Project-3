Visualizing Malaria's Effect on the World

In this project, we utilzed the Javascript library Leaflet to create an interactive map that shows the relationship between Malaria related deaths and GDP per capita. Our Leaflet maps' base layer shows GDP by color scale and provides statistics by age group upon clicking each country. In our second visualization, we used the Javascript library Charts to help display the global trend in Malaria deaths over the 2000s. Finally, we've utilized Plotly to highlight Malria vs. GDP in he most heavily effected countries. Click 'Maps & Charts' in our navigation or the images below to check out our data visualizations. Our project is backed by a Python script that feeds a CSV document (see data source above in navigation) into a PostgreSQL database and table.

---

According to the World Health Organization, there were approximately 219 million Malaria cases worldwide in 2017. These cases resulted in about 435,000 deaths.
