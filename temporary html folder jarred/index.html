Visualizing Malaria's Effect on the World

In this project, we utilzed the Javascript libraries Leaflet and amCharts to create interactive maps that show the relationships between Malaria related deaths and GDP per capita. Our Leaflet maps' base layers show GDP by color scale and provide statistics by age group upon clicking each country. Our amCharts map shows the total number of deaths per country with sized circles to help illustrate severity by country. We've also utilized Plotly to create a chart to better display the relationship between GDP and Malaria deaths. Click 'Maps & Charts' in our navigation or the images below to check out our data visualizations. Our project is backed by a Python script that feeds a CSV document (see data source above in navigation) into a PostgreSQL database and table.

---

According to the World Health Organization, there were approximately 219 million Malaria cases worldwide in 2017. These cases resulted in about 435,000 deaths.
